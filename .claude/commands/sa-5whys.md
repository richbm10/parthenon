---
description: Run a 5 Whys inquiry on the current topic or a new sub-topic.
---

Invoke the `software-analyst` sub-agent to conduct a focused 5 Whys inquiry.

The agent will:

1. State the topic under investigation
2. Begin WHY-1 from the last confirmed stakeholder statement
3. Run the chain to root cause (minimum 5 levels)
4. Produce a ROOT CAUSE statement

If a session is already in progress, continue from the last recorded Why level.
